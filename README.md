## Boilerplate

  A minimal seed comprising of Jade, Stylus, Gulp and some basic imports.

## Installation

```sh
# Clone this repository
$ git clone https://github.com/sasha-kazakevich/boilerplate-jade-stylus.git new_project
$ cd new_project

# install dependencies
$ npm install

# start
$ npm start

# open localhost:8000
```
